This is a project to study the faulty medical devices in the dataset from Kaggle: https://www.kaggle.com/datasets/vladimirmijatovic/faulty-medical-devices-global-dataset

The result of the project should be:
1- ETL data life of data.
2- EDA and cleaning using pandas and jupyterlab and preeliminar analysis
3- Upload clean data to a local database
4- Create a dashboard for the final result

The objective and motive of the exitence of this project is for educational purposes of the author of the analysis Arizael García Capote.
This a project of type Discovery.

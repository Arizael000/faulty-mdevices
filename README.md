Faulty Medical Devices Dataset Analysis Project
Overview
Welcome to our project, which aims to explore and analyze the dataset from Kaggle: Faulty Medical Devices Global Dataset. Our objective is to gain insights into the faulty medical devices across different regions and time periods.

Project Objectives
ETL Data Life Cycle: Extract, Transform, and Load data for comprehensive analysis.
Exploratory Data Analysis (EDA): Conduct in-depth analysis using Pandas and JupyterLab.
Data Cleaning: Perform preliminary cleaning to ensure the dataset is ready for further analysis.
Database Upload: Store clean data locally for easy access and querying.
Dashboard Creation: Develop a user-friendly dashboard to visualize our findings.
Educational Value
This project serves as a practical exercise in data science and machine learning. It aims to educate Arizael García Capote, the author of the analysis, about best practices in data preprocessing, exploratory analysis, and dashboard development.

Project Type
This is a Discovery project, designed to explore new datasets, gain insights, and develop essential skills in data analysis.
